# Yêu cầu Topic Seminar cho Project Search Học Thuật

## Context Project
Nhóm tôi đã thực hiện một project "Hybrid Search System for Academic Documents using Elasticsearch", gồm các thành phần chính:
- Chuẩn hóa/nhúng dữ liệu học thuật (arXiv > 1 triệu paper).
- Tìm kiếm kết hợp truyền thống (BM25) và semantic vector (sentence-transformers, kNN search, Elasticsearch dense_vector).
- Dùng Reciprocal Rank Fusion (RRF) để hybrid score.
- Có UI demo, benchmark, đánh giá thực nghiệm.

## Yêu cầu Topic Seminar
- **Thời lượng**: 15 phút, thuyết trình nhóm 3 người
- **Phong cách học thuật:** đề tài lý thuyết chuyên sâu nhưng liên hệ được với hệ thống thực tế
- **Nên tập trung vào một vấn đề/chủ đề thuật toán/phân tích, không chỉ là demo hệ thống tổng quát
- Đề xuất topic nên đáp ứng tiêu chí:
  - Có nền tảng lý thuyết sâu, ví dụ: một giải thuật IR, vector search, fusion, ranking, reranking, NLP, v.v.
  - Có tính ứng dụng, gắn với thực tế xử lý/triển khai search hệ thống lớn (như project nhóm tôi đã làm)
  - Có thể chia rõ 3 phần để mỗi thành viên trình bày logic.

## Một số keyword liên quan project
- Hybrid Search, Vector Search, BM25, kNN Search, Dense Vector, Sentence-Transformers, Reciprocal Rank Fusion (RRF), Fusion Methods, Ranking Algorithm, Scalability, Information Retrieval, Benchmark, Academic Search, Large-Scale Search System

## Mục tiêu khi hỏi AI
- Gợi ý 2-4 đề tài seminar có tính lý thuyết, sâu sắc và có thể trình bày theo nhóm 3.
- Ngoài đề tài, nếu có thể hãy gợi ý luôn outline chia phần/thành viên cho từng đề tài ấy.

**Vui lòng liệt kê ý tưởng topic theo tiêu chí trên, ưu tiên các đề tài mới mẻ hoặc có nền tảng toán/giải thuật tốt, ứng dụng thực tiễn rõ ràng!**
